# Required Images

Please add the following image files to this `public` folder:

1. **remi.png** - The Remi logo (used on welcome screen)
2. **elephant1.png** - Elephant mascot for welcome screen
3. **elephant2.png** - Elephant mascot for upload screen and dashboard (alternating)
4. **elephant3.png** - Elephant mascot for dashboard (alternating)
5. **elephant4.png** - Elephant mascot for done screen

All images should be PNG format and will be automatically loaded from the public folder.
